  • 腾讯验证码的十二年 : 腾讯的验证码发展之路, 和之前阿里的case一样; 都是在追求极致和卓越的过程; 验证码并不是一个大系统，但是任何事情只要做到极致，都有很多可以学习和深挖的点; 同时也找到了这篇文章neuroradiology/InsideReCaptcha具体介绍了Google ReCaptcha的验证码逆向解析，很是精彩

  • Apache Beam: 下一代的大数据处理标准 : “Apache Beam（原名Google DataFlow）是Google在2016年2月份贡献给Apache基金会的Apache孵化项目，被认为是继MapReduce，GFS和BigQuery等之后，Google在大数据处理领域对开源社区的又一个非常大的贡献”, Beam设计了统一的流式处理的API，其具体实现可以有Apache Flink等支持

  • Top-down learning path: Machine Learning for Software Engineers : 机器学习的入门到深入，在git上面得到不少好评，可以关注下

  • A Formal Security Analysis of the Signal Messaging Protocol : 对端到端加密协议Signal的安全性，可靠性的分析，感觉高大上，非常复杂；先mark吧

  • 警惕废话指标（Bullshit Metrics），关注唯一关键指标 : 同感
