weekly of 20161023

新鲜事

工具技巧

  • LINUX KERNEL NETWORKING PACKET RECEIVING PROCESS : linux对于网络包的处理流程说明；几个图画的非常的精细;

  • HOW BREAKPOINTS ARE SET : 其他的关于debuger的文章包括: Implementing a Debugger: The Fundamentals, How does gdb work?, gdb Debugging Full Example (Tutorial): ncurses

系统设计

  • CSP的今世与未来 : 具体介绍了CSP的核心功能特性，现在CSP level 2的不足之处: 维护成本太高; 以及最近引入的CSP level3的strict dynamic特性, 可以很好的在实践中做一个折衷; 贴吧也是一个非常容易遭受XSS攻击的站点，对XSS的防御长路漫漫

  • CacheP2P : 介绍基于web浏览器的p2p cache架构，感觉对于浏览器端的视频播放等可以起到节省带宽的作用

  • Low Tech Homomorphic Encryption : 文章描述了一个场景，就是在云计算的场景下，如何保证对不安全的第三方的安全性; 一个思路是以list的方式发送一堆数据，里面仅有部分是真正的数据；因此来达到混淆的目的

其他

  • the cryptopals crypto challenges : 关于加密相关的测试题，相信通过这些训练，可以让我们对加密算法有更深的理解

  • Hash functions: An empirical comparison : 听老的一个文章，关于hash 函数的比较; 作者首先说明用户hash table构建的hash 和普通的单向加密hash 函数要求是不一样的；最后通过对比，得出的结论是: “Murmur2, Meiyan, SBox, and CRC32 provide good performance for all kinds of keys. They can be recommended as general-purpose hashing functions on x86.”
