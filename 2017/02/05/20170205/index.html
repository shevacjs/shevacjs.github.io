weekly of 20170205

2017-02-05

新鲜事

  • The future of iOS is 64-bit only: Apple to stop support of 32-bit apps : 新的IOS系统即将不支持32位系统的程序

  • Integrating Progressive Web Apps deeply into Android : 谷歌宣布，PWA将获得与安卓原生应用同等的待遇与权限，用户体验将完全一致, 包括桌面图标，通知等支持。如同大家预料的，Chrome浏览器与安卓的底层正在逐渐打通，H5开发的前景越来越光明;

  • Reload, reloaded: faster and leaner page reloads : Chrome最近动作不断, 新的reload特性可以减少无效的网络请求(默认是304的静态请求)，大大提升页面的性能; 此特性在前一期的weekly 已经提及

工具技巧

  • Algorithm to dynamically monitor quantiles : P^2算法的介绍，可以支持动态，增量地，以固定存储空间去计算一个特意的分位点, 还是非常精妙的, 先收藏，后续看下如何落地; P2算法可以参见The P-Square Algorithm for Dynamic Calculation of Percentiles and Histograms without Storing Observations

  • Stop Disabling SELinux: A Real-World guide : 本文介绍了具体使用SELinux的用于访问控制的实例, 作者认为SELinux在实践中还是能发挥非常大的作用，建议我们保留

  • Introducing Lottie : Airbnb开源的跨系统的动画解决发个那你，和之前的facebook的keyframe很像, 也可以参看中文的介绍版本这个项目碉堡了

  • How to choose algorithms for Microsoft Azure Machine Learning : 个人觉得是对机器学习新手入门的一个非常好的文章, 包括对算法选择的考量，算法的具体分类和介绍；是对现在机器学习算法领域一个较高程度的概述

系统设计

  • Unlearning descriptive statistics : 本文介绍了在统计方法里面，一些容易被误解或者被遗漏的一些概念，比如说中位数和均值的意义，分位点的重要, 如何衡量分布的均衡性和奇异点, 另外hacker news上面的评论也是需要重点关注的; 包括一些建议可以去了解的Five-number summary用来描述一堆有序数据, 用Medcouple 来评估分布的均衡性, Modes, Medians and Means: A Unifying Perspective用统一的视角来定义均值，中位数; 以及Statistics for Engineers: Applying statistical techniques to operations data 这些纯统计的概念在具体工程应用里面的还是会发挥不少作用的

  • TLS 1.3 @ 33c3 by Filippo Valsorda : TLS 1.3相比于之前的TLS 1.2有较大的飞跃，主要体现在于协议之初的密钥交互过程，可以支持1-RTT, 甚至O-RTT

  • Beringei: A high-performance time series storage engine : Facebook之前提到的基于全内存的时间序列的数据库的开源实现的介绍

其他

  • Four Column ASCII : 以四列的方式显示ASCII表，会发现更多有趣好玩的情况

  • Think OS: A Brief Introduction to Operating Systems : 一个OS的入门教程，感觉外国人做这个东西做得非常的精细，特别赞

  • 评论：机器学习和深度学习的最佳框架大比拼 : 对于要入门深度学习的同学来说, 是一个不错的入门参考手册; 大概的结论是: 对于深度学习，或者TensorFlow是现阶段最好的选择, 虽然他的入门曲线有点抖

  • Hacker News Highlights : 有定义的Hacker News上面一些高回复/高赞成的文章合集, 可以收藏mark

  • Microsoft/DMTK : Microsoft Distributed Machine Learning Toolkit, 里面有不少干货，包括Microsoft/LightGBM 等, LightLDA
