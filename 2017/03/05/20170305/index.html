新鲜事

  • Annotation is now a web standard : 标注功能现在是web 浏览器的一个新标准，个人感觉内容分发等之类业务可以考虑进一步和此标准结合

工具技巧

  • Why can’t I send UDP packets from a browser? : 一个很简单的问题，就是浏览器是否会发出UDP请求呢? 本文介绍了浏览器发送UDP场景的一些case, 比如说QUIC, Websocket等, 本文具体介绍了浏览器支持的不同协议, 以及不同协议支持的业务场景

  • 网易乐得iOS无埋点数据SDK实践之路 : 贴吧这边是不是也可以调研考虑下?, 同样可以关注的还有美团点评前端无痕埋点实践

  • Prophet: forecasting at scale : Facebook开源的趋势预测库, 感觉接入挺方便的, 我们可以测试支持下, 另外同时也看到另外一个类似的工具Unplugg: An automated Forecasting API for timeseries data 同样是用来分析时序数据

  • /dev/shm介绍 : Linux下面的内存文件系统, 可能会有用

系统设计

  • 【论文每日读】Learning from User Interactions in Personal Search : 一个很新颖的想法, 不过部分东西不是很能理解, 明天再好好研究下

  • A cartoon intro to WebAssembly : 一些系列文章, 对WebAssembly 有非常详细的介绍和说明; 个人感觉WebAssembly是不是可以应用于我们一些可行前端的构建和支持, 期待其快速的标准化和浏览器的支持

  • In search of a simple consensus algorithm : 对一个一致性协议的探讨, 作者认为Single Decree Paxos is simpler than Multi-Paxos & Raft , 一些论证细节下来再研究下，先mark

  • Shopify的闪购限流解决方案 : 一些限流方案&思路值得学习, 很多思想和我们线上的反攻击验证码类似, 包括可以注意和进一步总结的Secure Session Cookies

其他

  • Save time by transforming images in the command line : 文章介绍了使用ImageMagick 这个工具, 在命令行环境下面进行图片处理和切换的一些技巧, 可以试试

  • Hackernews Weekly : 该网站收集每周hacker news的热门文章, 如果精力有限，可以看看这里的内容

  • Top 3 machine learning libraries for Python : 三个关键字Theano, TensorFlow 和scikit-learn

  • Nonce misuse resistance 101 : 先mark

  • raft 经典场景分析 : mark

  • Some cases of insecure NGINX configurations : 挺有意思的, 一些配置的trick点; 可以关注下
