新鲜事

  • The Google Brain team — Looking Back on 2016 : Google Brain的summary，比较关注的是他们的政策: Spreading Machine Learning within Google

工具技巧

  • Introducing Draco: compression for 3D graphics : google自己设计的Draco压缩算法, “can be used to compress meshes and point-cloud data”

  • 10 examples of Linux ss command to monitor network connections : 最近发现linux下面的ss命令很强大，对观察socket状态非常有帮助, 比如看socket的cwnd，rto, rtt等

系统设计

  • Google Infrastructure Security Design Overview : 关于google安全体系建设的概述性文章, 从内部的存储，RPC交互到最上层的网络接入等，整体思路是非常体系和系统的; 关于service的相互认证, 可以考虑下,包括内部RPC交互需要携带用户认证信息, 以便进行内部加密和保护, 对贴吧也许也有不少参考意义;

  • 亿级用户PC主站的PHP7升级实践 : 微博的PHP7落地实践，HugePage的问题好像不少公司都遇到过

  • Enabling HTTP/2 for Dropbox web services: experiences and observations :Dropbox分享了他们迁移http2的经验，直观的收益/问题包括包体减小(采用HPACK格式), 新的调试体验/工具

  • Security Through Transparency : 关于Key Transparency的概念和想法，现在还不是很清楚，先mark，参考链接包括: CONIKS, Hacker News讨论

  • How Hash Algorithms Work : Hash算法的入门介绍，排版和内容组织方式都很好
