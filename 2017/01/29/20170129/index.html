weekly of 20170129

新鲜事

  • APFS is coming soon: iOS 10.3 will automatically upgrade your filesystem : IOS 10.3会自动将你的文件系统升级为APFS, 后者为苹果去年推出的新的基于SSD存储的文件系统

工具技巧

  • 千兆网卡真的能每秒发100万个包么？ : 结论是不能；因为最小的每个包大小(MSS)在不同的平台都有限制，比如linux, min(MSS) = 88 Byte, 意味着千兆网卡每秒能发送125Mbps / 188 B = 0.88 Mpps

  • Rules of Machine Learning: Best Practices for ML Engineering : google的机器学习的最佳工程实践，可以好好关注下

系统设计

  • Building Zero protocol for fast, secure mobile connections : 我们知道基于TLS的https协议交互之前总是需要1~2次RTT，用于密钥的交互; 本文介绍了facebook听过引入QUIC crypto的方法，以便达到0-RTT的效果，这些工程实践也会反馈给TLS 1.3协议(TLS 1.3协议设计上面就支持0-RTT，我们可以拭目以待)

  • How Airbnb manages to monitor customer issues at scale : Airbnb自己的用户反馈处理平台，基于ES的存储，已经自动的趋势预测和发现

  • Using Immutable Caching To Speed Up The Web : Mozila的博客介绍facebook通过引入Cache-Control: immutable来减少无必要的304请求, 提升整体页面的性能; 几个点可以注意下:1) 之前一直认为如果本地有未过期的缓存, 浏览器是不会发请求给server，但是按照这篇文章的说法, 当我们主动点击reload按钮, 就会无视本地的缓存去请求server的, 2)通用根据上面那篇文章，我们可以知道不同浏览器对避免无效刷新的请求方法各异，比如chrome主要是依靠自己的算法策略来实现的, 也可以看下facebook专门介绍的这篇文章This browser tweak saved 60% of requests to Facebook

  • Linux Performance : Brendangregg 关于linux performance监控和发现的资料合集，材料非常多, 大家可以收藏 & 慢慢学习下

其他

  • Introduction to High-Performance Scientific Computing : 看起来像是不错的材料，先mark

  • Python 为何能坐稳 AI 时代头牌语言 : 对于一个有个C背景的程序员而言，一直不喜欢python的代码风格和语法; 不过最近对机器学习的领域的了解，python的确是当之无愧的AI语言, 文章论述了为什么其可以拥有如此地位: “Python 是众多主流语言中唯一一个战略定位明确，而且始终坚持原有战略定位不动摇的语言”

  • Linux 升级内核开启 TCP BBR 有多大好处 : 本文以具体的示例说明了TCP BBR拥塞算法的优缺点, 对与那些有一定丢包率的网络(广域网)，BBR的优势是比较明显的

  • Site Reliability Engineering : 可以免费在线观看的Google SRE的书, 虽然我好像有PDF版本
