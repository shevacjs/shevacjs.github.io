新鲜事

  • Chrome 59 has cross-platform headless support : Chrome开始支持headless模式，这对PhantomJs等造成一定的冲击和影响, 可以参考帮助文档Headless Chromium

工具技巧

  • 在Elasticsearch中应用机器学习排序LTR : Es的一个插件，可以自定义排序，对于推荐等业务场景有较大帮助

系统设计

  • A Comprehensive Guide To HTTP/2 Server Push : Http2 Server Push机制的非常全面和详细的介绍, 现在其server push的机制和我们直观理解上面的有所偏差，更多是指”资源的主动推送”, 这样可以加快整个首屏的渲染时间, 文章还具体介绍了不同技术栈的配置情况, 不过nginx现在默认不支持

  • 微信高可用分布式数据库PhxSQL设计与实现 : 可以学习了解下, 设计还是很精确的

  • 干货 | 携程实时用户行为系统实践 : 参考下

  • 反爬虫和抗DDOS攻击技术实践 : 文章说的不是很清楚，但是反spider的工作还是需要去做的

  • Stop using JWT for sessions : 关于JWT的一些说明, 作者建议不要用JWT的机制作为单点登录验证的机制, 还可以参看JSON Web Token - 在Web应用间安全地传递信息 和八幅漫画理解使用JSON Web Token设计单点登录系统
